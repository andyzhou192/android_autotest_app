# android_autotest_app
使用AndroidTestRunner进行SDK测试的框架，依赖模拟器或真机，适用于Android9以下的版本

本代码使用的Android Studio 3.5.1  + Gradle 4.1 + Android Gradle Plugin 3.2.0环境开发
